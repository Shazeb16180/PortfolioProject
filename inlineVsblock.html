Inline and Block Elements

July 18,2021

Introduction

Every HTML element has a default display value, depending on what type of element it is.There are two display values: block and inline.

Description

Block-level Elements:

The elements that begin on a new line are known as block elements. A block element takes up the full width of the content. Unlike inline, there exists a top and bottom margin for these elements. Block-level elements may only appear inside the body tag. Block-level elements create a larger structure than inline elements.Some of the Block Elements are:

  • div
  • p
  • li
  • main

Inline-level Elements

Inline as the name says “included as a part of the main text and not as a separate section”. Inline elements occupy the space as needed within the space defined by the main element. Unlike block-level elements, they do not begin on new lines.Some of the Block Elements are:

  • a
  • input
  • span
  • img

Inline-Block Elements

Alright, let's move on to inline-block. It’s essentially the same thing as inline, except that you can set height and width values

conclusion

  • Inline:The element doesn’t start on a new line and only occupy just the width it requires. You can’t set the width or height.
  • Inline-block: It’s formatted just like the inline element, where it doesn’t start on a new line. BUT, you can set width and height values.
  • Block: The element will start on a new line and occupy the full width available. And you can set width and height values.
