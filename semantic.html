Semantic tags

July 18,2021

Introduction

HTML5 introduced a consistent list of semantiic ELements to help search engines and developers.HTML5 semantic tags define the purpose of the ELement.

Description

Semantic elements have meaningful names which tells about type of content. For example header, footer, table, â€¦ etc. HTML5 introduces many semantic elements as mentioned below which make the code easier to write and understand for the developer as well as instructs the browser on how to treat them.some of them are:

  • article
  • aside
  • detaials
  • figcaption
  • figure
  • section
  • header
  • footer

Benefits

  1. Search engines will consider its contents as important keywords to influence the page's search rankings (see SEO)
  2. Screen readers can use it as a signpost to help visually impaired users navigate a page
  3. Finding blocks of meaningful code is significantly easier than searching through endless divs with or without semantic or namespaced classes
  4. Suggests to the developer the type of data that will be populated.
  5. Semantic naming mirrors proper custom element/component naming.
